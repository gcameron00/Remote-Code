About This Collection

A personal catalogue of physical music media — spanning six decades of records, tapes, and discs across every genre. Browse the collection, explore the artwork, and dig into the details of each release.

How to Browse

The main catalogue page offers two ways to explore the collection:

  • Grid view — a responsive card layout showing cover art and key metadata. Click any card to open the full detail view with tracklist and notes.
  • Cover Flow — a 3D cover-art browser inspired by Apple's CoverFlow. Navigate with the arrow buttons, keyboard arrow keys, or by swiping/dragging. Click the centre cover to open the detail view.

Use the filter panel on the left to narrow the collection by media type, genre, or condition, and the search bar to find a specific artist, title, or label.

Media Types

The collection covers the full spectrum of physical formats:

LP CD Cassette 12-inch 7-inch MiniDisc

Condition Grading

Records are graded using the standard collector scale:

  • Mint — Unplayed, perfect condition.
  • VG+ — Minimal signs of play; plays perfectly.
  • VG — Some light marks; plays with minimal noise.
  • Good — Noticeable marks; playable but audible wear.
  • Poor — Heavy wear; for display or parts only.

Admin & Data

The admin panel allows the collection owner to add, edit, and delete records. Changes are stored locally in the browser (demo mode). The default admin PIN is 1234.

Cover images are auto-generated from each record's artist and title if no image URL is provided, giving each release a unique, consistently-styled placeholder.
